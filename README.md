# msgpack.cr

A low-level [msgpack](http://msgpack.org) codec for [Crystal](https://crystal-lang.org)

## TODO

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  msgpack:
    github: steakknife/msgpack.cr
```
## Usage

```crystal
require "msgpack"
```
